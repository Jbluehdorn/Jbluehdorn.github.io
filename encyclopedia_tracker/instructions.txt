I want to create an app that can parse image data from an uploaded screenshot and read a user's animal crossing encyclopedia data. 
Then it should parse and save that data to local storage. 
It should then show them all the available fish, insects, and sea creatures that they can catch based on the date and time. 

Before you start, you should know that our base data to work from is all listed in the json directory of this project.

==Workflows==

Upload
1. User uploads a photo from their phone of their in-game encyclopedia. 
2. We cross-reference that with our json data to map all the encyclopedia entries we see.
3. That information gets saved to localStorage for further user

Display
1. We check the date and time of the user to get a base of where to start
2. We check through our data to see what is available that they haven't already collected
3. We show them all the available 
4. We allow the user to change the date, time, and hemisphere of the user to account for users that have adjusted their times manually
